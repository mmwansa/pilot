About VACMS

Verbal Autopsy & Case Management System (VACMS) is a web application built to support the Sample Registration System (SRS) project at the Zambia National Public Health Institute (ZNPHI). This project is a fork of the VA Explorer (VAE) project, licensed by The MITRE Corporation under Apache 2.0.

Key Functionality

  • User account management, including:
    • Creation/disabling of user accounts by administrators
    • Password management for individual users
  • User access controls, including:
    • Role-based access with the following roles: administrators, data managers, data viewers, and field workers
    • Assignment to one or more geographic areas for geographical-level scoping of data
  • Case management and follow-up, including event tracking and VA scheduling
  • Loading of event and verbal autopsy questionnaire data
  • Assignment of cause of death using InterVA5 algorithm (Note: relies on external pyCrossVA and InterVA5 services)
  • Exploration of cause of death data via a dynamic, visualization-based dashboard that includes:
    • A dynamic heatmap showing geographical trends, with zoom capabilities to hone in on regions of interest
    • Cause of death plots for chosen regions
    • Death distributions by age, gender, and place of death for chosen regions
    • Trends over time for chosen regions
